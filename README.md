# zk-dev3pack

Notas de estudo do **ZK & Privacy Bootcamp - April 2026** (dev3pack), ministrado pela Dra. Melissa (0xMilica).

Cada pasta corresponde a uma sessão e contém:

- `README.md` — notas estruturadas, comentários técnicos/correções e exercícios (mix conceitual + prático)
- `transcript-raw.md` — transcrição automática crua da aula (extraída via NotebookLM)
- `slides-*.{pdf,pptx}` — apresentação da instrutora
- `reading-week*.pdf` — leitura semanal recomendada
